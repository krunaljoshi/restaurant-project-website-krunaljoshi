Week #1 Questions and Answers

HTML Question

Q. Do all the tags come in pair?
Ans: No, There are tags that come in single tags as well. eg: < br > The Break Line Tag, < hr > tag that creates a line.

CSS Question

Website Structure Question 1

Q. How is the main page of the Website divided in HTML?
Ans: Main content is divided in < main > with various subsections represented by < article >, < section > and < div >. Every page has one < main > tag.

Website Structure Question 2

Q. What does < header > represent?
Ans: It represents a group of introductory content, it represents header for the webpage depending on it's position. header tag can be placed and defined accordingly.
